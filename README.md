# License plate
<p>淘然视界车牌号数据（如有问题，可以随时在Issues中提问）：</p>
<p>1）数据规模：当前1000，持续更新</p>
<p>2）数据种类：图片</p>
<p>3）采集设备：卡口相机</p>
<p>4）采集场景：室外交通卡口</p>
<p>5）角度：卡口标准角度</p>
<p>6）光照分布：多样</p>
<p>7）分辨率：1600*1200 </p>
<p>8）数据格式：jpg，png</p>
<p>9）应用：公安、安防车辆分析及停车场自动识别</p>
